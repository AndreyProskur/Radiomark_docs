Studies

Управление медицинскими снимками

Создать новый снимок или отправить исправленный на модерацию

Если в теле запроса передан uploadId, создаётся новый снимок из временного файла Если передан studyId, обновляется существующий снимок

Authorizations:
BearerAuth
Request Body schema: application/json
required
uploadId
string

Идентификатор временного файла (для нового снимка)

studyId
string

Идентификатор существующего снимка (для исправления возвращённого)

modality
required
string (Modality)
Enum: "X-Ray" "MRI" "CT"

Тип исследования

region
required
string (Region)

Область исследования

description
required
string

Описание снимка

conclusion
required
string

Заключение врача

Array of objects (AnnotationRequest)

Размеченные структуры

mode
string
Enum: "T1" "T2" "PD" "FLAIR"

Режимы МРТ

contrast
string
Enum: "None" "Oral" "IV" "Oral-IV"

Тип контрастного вещества

sliceThickness
number [ 0.1 .. 30 ]

Толщина среза в мм

Responses

https://radiomark.com/api/v1/studies/submissions

Request samples

  • Payload
Content type
application/json
{
  • "uploadId": "string",
  • "studyId": "string",
  • "modality": "MRI",
  • "region": "Lungs",
  • "description": "Наличие области затемнения в верхней доле правого лёгкого",
  • "conclusion": "У пациента выявлена пневмония верхней доли правого лёгкого",
  • "annotations": [
    • {
      • "name": "Правое легкое",
      • "coordinates": {
        • "x": 120,
        • "y": 80,
        • "width": 200,
        • "height": 150
        }
      }
    ],
  • "mode": "T2",
  • "contrast": "None",
  • "sliceThickness": 2
}

Response samples

  • 201
Content type
application/json

  • "studyId": "SN-041",
  • "status": "pending",
  • "submittedAt": "2026-04-15T11:45:00Z"
}
