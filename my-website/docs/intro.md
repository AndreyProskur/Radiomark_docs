_Radiomark_ — образовательная база размеченных рентгенологических снимков, сервис ускоренной обработки снимков и получения заключений для пациентов, а также источник качественных датасетов для обучения ИИ.

:::info[Карточка сервиса]

Версия: **1.0.0** · Среда: **Production** · Обновлено: **2026‑05‑04**

:::

## Основные функции

- **Загрузка и разметка медицинских снимков** — DICOM, КТ, МРТ, рентген с графическими аннотациями.
- **Просмотр и тренировка на учебных снимках** — доступ к базе размеченных изображений для студентов.
- **Модерация и контроль качества** — двухэтапная проверка разметки и описания.
- **Заявки от пациентов** — создание заявки, назначение врача, получение заключения.
- **Регистрация и профили** — единая учетная запись с ролевой моделью.
- **Экспорт датасетов** — выгрузка обезличенных размеченных данных для ML‑разработки.

- Доступность сервиса > **99.0%** в месяц. Допустимый простой — не более **7.2 часов** в месяц.
- Время ответа API (p95) ≤ **500 мс** при нагрузке до 500 RPS.

:::
