# Radiomark — платформа для разметки медицинских снимков

Платформа Radiomark предназначена для врачей-рентгенологов, студентов медицинских вузов и пациентов. Она обеспечивает полный цикл работы с медицинскими изображениями:

- загрузка снимков (рентген, КТ, МРТ)
- разметка структур и описание
- модерация и контроль качества
- формирование медицинских заключений
- тренировка студентов на реальных кейсах

Документация включает архитектурные решения, описание API, модель данных и пользовательские сценарии.
