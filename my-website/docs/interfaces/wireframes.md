# Макеты интерфейса

В этом разделе представлены низкодетализированные макеты (wireframes) основных экранов платформы. Для каждого экрана указаны задействованные API-эндпоинты.

## Экран 1. Загрузка медицинского снимка

Пациент или врач заполняет форму загрузки: выбирает тип и область исследования, загружает файл, добавляет краткое описание. При необходимости можно изменить метаданные ранее загруженного снимка или удалить временный файл.

![Макет экрана загрузки снимка](/img/Upload_screen.png)

<details>
<summary>📋 API-эндпоинты экрана загрузки</summary>

| Действие | Метод | Endpoint |
|:---------|:------|:---------|
| Загрузить новый снимок | `POST` | `/api/v1/studies/uploads` |
| Обновить информацию о загруженном снимке (при внесении изменений) | `PATCH` | `/api/v1/studies/uploads/{uploadId}` |
| Удалить временный файл | `DELETE` | `/api/v1/studies/uploads/{uploadId}` |

</details>

## Экран 2. История снимков и их статусы

Врач видит список всех загруженных снимков с фильтрацией по статусу, модальности и области исследования. При выборе снимка отображаются его детали.

![Макет экрана истории снимков](/img/History_screen.png)

<details>
<summary>📋 API-эндпоинты экрана истории</summary>

| Действие | Метод | Endpoint |
|:---------|:------|:---------|
| Получить список всех снимков | `GET` | `/api/v1/studies` |
| Отфильтровать снимки по статусу, модальности, области | `GET` | `/api/v1/studies?status={status}&modality={modality}&region={region}` |
| Получить детали конкретного снимка | `GET` | `/api/v1/studies/{studyId}` |

</details>

## Экран 3. Разметка и описание медицинского снимка

Врач выполняет графическую разметку структур на снимке и пишет заключение. С нового снимка можно отправить на модерацию, для возвращённого — исправить и отправить повторно.

![Макет экрана разметки снимка](/img/Work_screen.png)

<details>
<summary>📋 API-эндпоинты экрана разметки</summary>

| Действие | Метод | Endpoint |
|:---------|:------|:---------|
| Получить полную информацию о существующем снимке | `GET` | `/api/v1/studies/{studyId}/full` |
| Получить список размеченных структур | `GET` | `/api/v1/studies/{studyId}/annotations` |
| Добавить новую структуру разметки | `POST` | `/api/v1/studies/{studyId}/annotations` |
| Отправить новый снимок на модерацию | `POST` | `/api/v1/studies/{uploadId}/submissions` |
| Показать изображение и данные загруженного снимка | `GET` | `/api/v1/studies/uploads/{uploadId}` |

</details>
