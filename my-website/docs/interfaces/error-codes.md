# Коды ошибок

## HTTP-статусы

| Код | Статус | Когда возникает |
|:----|:-------|:----------------|
| 200 | OK | Успешное получение или обновление ресурса |
| 201 | Created | Успешное создание ресурса |
| 204 | No Content | Ресурс успешно удалён (нет содержимого в ответе) |
| 400 | Bad Request | Ошибка валидации: отсутствуют обязательные поля или неверный формат |
| 401 | Unauthorized | Неавторизован – требуется JWT‑токен |
| 404 | Not Found | Ресурс с указанным идентификатором не найден |
| 409 | Conflict | Конфликт состояния – см. коды ошибок ниже |
| 413 | Payload Too Large | Файл слишком большой (максимум 100 МБ) |
| 415 | Unsupported Media Type | Неподдерживаемый формат файла |
| 500 | Internal Server Error | Внутренняя ошибка сервера |

## Коды ошибок в теле ответа

При ошибке API возвращает JSON с полями `code` и `message`:

**Пример ответа 409**
```json
{
  "code": "STUDY_ALREADY_SUBMITTED",
  "message": "Снимок уже отправлен на модерацию"
}
```
### Справочник кодов

| Код ошибки | HTTP | Описание |
|:-----------|:-----|:---------|
| `VALIDATION_ERROR` | 400 | Не переданы обязательные поля или нарушены ограничения |
| `STUDY_NOT_FOUND` | 404 | Снимок с указанным `studyId` не найден |
| `UPLOAD_NOT_FOUND` | 404 | Временный файл с указанным `uploadId` не найден или истёк |
| `ANNOTATION_NOT_FOUND` | 404 | Размеченная структура не найдена |
| `STUDY_ALREADY_SUBMITTED` | 409 | Снимок уже отправлен на модерацию |
| `STUDY_NOT_RETURNED` | 409 | Снимок не находится в статусе `returned`, исправление невозможно |
| `FILE_TOO_LARGE` | 413 | Превышен максимальный размер файла (100 МБ) |
| `UNSUPPORTED_FORMAT` | 415 | Неподдерживаемый формат файла (допустимы DICOM, PNG, JPG) |
